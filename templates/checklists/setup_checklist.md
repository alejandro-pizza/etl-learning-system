# Setup Checklist

Use this checklist when first implementing the ETL Learning System.

## Physical Materials

- [ ] **Notebook 1** for raw capture (A5 / portable size recommended) 
- [ ] **Notebook 2** for refined Cornell notes (B5 / larger format recommended)
- [ ] **Pens** that you enjoy writing with
- [ ] **Sticky tabs** for marking pages
- [ ] **Optional:** Highlighters (1-2 colors max)

## Write in Notebooks

### Raw Capture (Inside Front Cover)
- [ ] Symbol reference
- [ ] Header format
- [ ] "After processing: Cross out date"

### Cornell (Inside Front Cover)
- [ ] Cornell method template
- [ ] Three-section layout guide

## Digital Setup

- [ ] **Create learning repository** on GitHub
- [ ] **Install Jupyter**: `pip install jupyter`
- [ ] **Copy templates** to your repo
- [ ] **Set up directory structure**
```
  courses/
  projects/
  exercises/
  templates/
  docs/
  resources/
```

## Test Run

- [ ] **Capture one day** of notes in Capture Notebook
- [ ] **Process one page** to test workflow
- [ ] **Create one notebook** from template
- [ ] **Update one doc page**
- [ ] **Commit to GitHub**

## Weekly Routine

- [ ] **Print weekly checklist** (or write in notebook)
- [ ] **Choose processing day** (Sunday or Monday)
- [ ] **Set 90-min calendar block** for processing
- [ ] **First processing session** completed

## Adjustments

After first week, adjust:
- [ ] Symbol set (if needed)
- [ ] Processing time/day
- [ ] Template structure
- [ ] Anything that felt clunky

---

Remember: Start simple, adjust as you go!
