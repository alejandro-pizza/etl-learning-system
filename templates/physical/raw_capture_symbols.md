# Raw Capture Symbols

Write this on the inside front cover of your A5 notebook.

## Symbol Reference
```
SYMBOLS:
- Note / Observation
□ Task / Action item
? Question / Unclear concept
* Important / Exam-relevant
> Code snippet / Example
! Deadline / Time-sensitive

HEADER FORMAT:
[DATE] | [TYPE] | [SOURCE]

Types: LECTURE, READING, VIDEO, MEETING, TASK
```

## Usage Examples
```
2024-12-14 | LECTURE | MLOps - Week 3
───────────────────────────────────────
- Docker containers isolate applications
□ Install Docker Desktop
? How does port mapping work exactly?
* Exam topic: container lifecycle
> docker run -d -p 8000:8000 myapp
! Project due Friday 5pm
```

## Tips

- **Consistent use** matters more than perfect use
- **Mark as you write**, don't add symbols later
- **Don't overthink** - when in doubt, use `•`
- **Cross out date** when page is processed
