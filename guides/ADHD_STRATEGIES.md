# ADHD Strategies Guide

How the ETL Learning System specifically addresses ADHD challenges and executive function difficulties.

---

## Understanding ADHD & Learning

### Executive Function Challenges

**ADHD affects:**

**Working Memory:**
- Difficulty holding information temporarily
- Forgetting what you were about to do
- Losing track of multi-step processes
- Hard to remember instructions

**Task Initiation:**
- Hard to start tasks (even enjoyable ones)
- Procrastination on important work
- Waiting for "perfect" moment that never comes
- Analysis paralysis

**Sustained Attention:**
- Easily distracted by environment
- Mind wanders during lectures
- Difficulty staying focused on reading
- Hard to complete long tasks

**Organization:**
- Scattered notes and materials
- Difficulty categorizing information
- Lost or misplaced items
- Chaotic systems that break down

**Time Management:**
- Poor time estimation
- Deadline panic
- "Time blindness" - losing track of time
- Difficulty prioritizing

**Emotional Regulation:**
- Frustration when systems don't work
- Overwhelm from complexity
- Anxiety about falling behind
- Rejection sensitivity

## Weekly ADHD Strategies

### Processing Session Setup

**Environment Optimization:**
```
Before Starting:

Physical Setup:
- Clean, clear desk
- Phone in other room (or airplane mode)
- Water bottle filled
- Snacks available
- Comfortable seating

Digital Setup:
- Close all browser tabs
```

**Mental Preparation:**
```
Mindset:
```

### During Processing

**Managing Focus:**
```
If Focus Wanes:

- Longer break (5 min)
```

**Managing Overwhelm:**
```
If Feeling Overwhelmed:

3. What's the NEXT step? (just one)
4. Do only that step
5. Then reassess

One step at a time.
```

**Managing Perfectionism:**
```
If Stuck on Perfect:

- Set 10-min timer for this section
- When timer goes off, move on
- "Good enough" IS good enough
- Can always refine later
- Done > Perfect

Perfectionism is procrastination in disguise.
```

## Specific ADHD Accommodations

### For Primarily Inattentive Type

**Challenges:**
- Lost in thought during lectures
- Miss important points
- Hard to filter important vs unimportant

**Strategies:**
```
During Capture:
- Use * symbol liberally (mark anything that might be important)
- Record lectures (if allowed) for gaps
- Write continuously to stay engaged
- Arrows to connect wandering thoughts

During Processing:
- Review * symbols first (likely important)
- Compare with syllabus/slides (catch missed content)
- Okay to have incomplete notes
- Supplementary reading can fill gaps
```

### For Primarily Hyperactive Type

**Challenges:**
- Fidgeting during study
- Need movement
- Restlessness during processing

**Strategies:**
```
During Capture:
- Standing desk or walk while listening
- Fidget toy in non-writing hand
- Doodles in margins
- Move positions frequently

During Processing:
- Standing desk
- Sit on exercise ball
- Frequent short breaks
- Background music/white noise
- Movement breaks every 20 min
```

### For Combined Type

**Challenges:**
- Both attention and hyperactivity

**Strategies:**
```
Use both sets of strategies above, plus:

- Shorter, more frequent sessions
  (3x 30-min instead of 1x 90-min)

- Pomodoro technique
  (25 min work, 5 min break)

- Accountability buddy
  (co-working session)

- Environmental change
  (coffee shop, library, different room)
```
